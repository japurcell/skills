In this skill version, Phase 3 stores the exploration outputs as local workflow artifacts, not as GitHub artifact subissues. Write the exploration summary to `02-exploration/summary.md`, write `files.csv` to `02-exploration/files.csv`, and write the open-questions ledger to `02-exploration/open-questions.md`. Then create a `phase:exploration` GitHub child issue and attach it to the Phase 1 parent with `gh api graphql ... addSubIssue`; that issue links to `02-exploration/summary.md`, so another agent resumes from the saved local files plus the GitHub sub-issue linkage.
