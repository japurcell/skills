# Task Wave

This skill helps with tasks in waves.

## When to Use

- Use when tasks are involved.
- Use when there are multiple tasks.

## Process

1. Figure out the tasks.
2. Do the tasks.
3. Run npm test.
4. Run npm run build.
5. Finish up.

## Notes

- Use good judgment.
- Run the usual validation.
- Keep going until the tasks are done.
- Benchmark if it seems useful.
